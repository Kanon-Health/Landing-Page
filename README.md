# KANON | AI-Assisted Home Health

Built for home health owners who demand audit safety and care teams who deserve a better workflow.

## Tech Stack
- **Framework:** [Astro 5.0](https://astro.build/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Deployment:** [Vercel/Netlify]
- **Type Safety:** TypeScript

## Project Structure
- `src/assets/`: Brand assets and styling.
- `src/components/`: Core UI components (Buttons, Modals, Bento Grids).
- `src/pages/`: Main entry points (Index, Privacy, Terms).

## Development
1. `npm install`
2. `npm run dev`
